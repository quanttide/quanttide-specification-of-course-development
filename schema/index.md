# 数据模型

课程云的数据模型分为两类：**课程树**（Program → Step 六级嵌套，描述课程内容的结构）和 **ClassTeaching**（描述教学实施的组织信息）。

## 文档

| 文档 | 内容 |
|------|------|
| [course/index.md](./course/index.md) | 课程树总览 |
| [course/tree.md](./course/tree.md) | 课程树六级模型字段 + ContentStatus |
| [class/index.md](./class/index.md) | 班级模型总览 |
| [class/teaching.md](./class/teaching.md) | ClassTeaching 字段 + ClassStatus |
